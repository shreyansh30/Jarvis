## üöÄ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/shreyansh30/Jarvis.git
   cd Jarvis
   ```

2. **Install required modules**
   ```bash
   pip install pyttsx3
   pip install wolframalpha
   pip install wikipedia
   pip install speechrecognition
   pip install pyaudio
   ```

   Or install all dependencies at once:
   ```bash
   pip install pyttsx3 wolframalpha wikipedia speechrecognition pyaudio
   ```

## üéØ Usage

1. Navigate to the project directory
2. Run the main Python file:
   ```bash
   python jarvis.py
   ```
   *(Note: Replace `jarvis.py` with your main file name if different)*

3. Speak your commands clearly into the microphone
4. Wait for Jarvis to process and respond

## ‚ö†Ô∏è Troubleshooting

- **Microphone not working**: Ensure your microphone is properly connected and configured
- **PyAudio installation issues**: 
  - On Windows: You may need to install PyAudio from a wheel file
  - On Linux:  Install PortAudio first using `sudo apt-get install portaudio19-dev`
  - On Mac: Install PortAudio using `brew install portaudio`
